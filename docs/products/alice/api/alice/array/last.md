---
### 定義
名前空間:Alice / アセンブリ : Losetta.dll / サポート: AliceScript1

現在の配列の最後の要素を表します

```cs title="AliceScript"
namespace Alice;
readonly variable Last;
```

### 例
以下は、`1`、`2`、`3`がある配列から最後の数を取得します。

```cs title="AliceScript"
var a = [1,2,3];
print(a.Last); // 出力例 : 3
```
